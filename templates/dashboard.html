Teach-On Dashboard

운영 상태, 최근 결과물, 멀티 에이전트 실행 흐름, 외부 프로그램/에이전트 연결을 한 화면에서 관리하는 콘솔입니다. 메인 제작 플로우는 유지하면서도 운영성과 확장성을 확보하는 용도에 맞춰 설계했습니다.

메인 제작 화면
운영 개요
최근 생성 결과와 실행 중인 작업, 산출물 수, 연결된 외부 커넥터 수를 종합해서 보여줍니다.
대시보드 데이터를 불러오는 중입니다...
보안 / 운영 설정
업로드 제한, 관리자 토큰, 헤더 보호, OCR/런타임 상태를 점검합니다.
대시보드 인증 상태를 확인하는 중입니다...
관리자 로그인
운영자 세션 로그인을 사용하면 브라우저 단위로 대시보드를 제어할 수 있습니다.
API / 자동화 토큰
외부 스크립트나 자동화 워커가 대시보드 API를 호출할 때는 헤더 토큰을 저장해 둘 수 있습니다.
보안 상태를 불러오는 중입니다...
접근 보호
관리자 로그인 또는 저장된 관리자 토큰으로 인증하면 운영 지표, 실행 중 Job, 외부 커넥터 목록을 볼 수 있습니다.
작업 모니터링
현재 분석 중인 job과 최근 생성된 강의안/시험지 결과를 함께 봅니다.
실행 중 / 최근 분석 작업
분석 작업을 불러오는 중입니다...
최근 생성 결과물
결과물을 불러오는 중입니다...
외부 프로그램 / 에이전트 연결
Webhook, 외부 Agent API, 자동화 서버를 연결해 Teach-On과 함께 쓸 수 있도록 확장합니다.
커넥터 목록
등록된 커넥터가 없습니다. 오른쪽 폼에서 새 커넥터를 추가해 보세요.
커넥터 등록 / 수정
다른 AI 에이전트 서버, n8n, Make, 사내 API, 자동화 워커를 등록할 수 있습니다. 우선 v1은 HTTP 기반 연동에 집중합니다.
